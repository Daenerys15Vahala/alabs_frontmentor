# QR Code Component

## Overview

This project is a solution to the **QR Code Component Challenge**. The goal was to build a responsive QR code card that closely matches the provided design using HTML and CSS.

The project focuses on creating a clean, centered layout while practicing fundamental front-end development skills such as semantic HTML, CSS styling, spacing, and responsive design.

## Features

- Responsive QR code card layout
- Semantic HTML5 structure

- Building responsive layouts that adapt to different screen sizes
